Personality × Drug Use
Proposal Dataset Visualizations Results

Dataset

Source

Fehrman et al. (2015), Drug Consumption (UCI). 1,885 respondents; demographics, OCEAN, impulsivity (BIS-11), sensation seeking (ImpSS), and self-reported usage for 18 drugs.

Access: Kaggle — Drug Consumptions (UCI), https://www.kaggle.com/datasets/obeykhadija/drug-consumptions-uci

Key Variables

  • Age, Gender, Education, Country, Ethnicity
  • OCEAN scores (NEO-FFI-R)
  • Impulsivity (BIS-11), Sensation Seeking (ImpSS)
  • Drug usage (18 items), levels CL0–CL6

Preparation

  1. Recode CL0–CL6 to binary user/non-user and ordinal 0–6
  2. Define drug groups: legal vs. illegal
  3. Clean labels, factor levels, and codebook alignment
  4. Create reproducible R scripts for wrangling and EDA

Reproducibility Notes

  • R Project with data/, code/, figs/, docs/
  • R Markdown report rendered to docs/final_report.html (linked from Results)
  • Use relative paths, set seed, and session info
